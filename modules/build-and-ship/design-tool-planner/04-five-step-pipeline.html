01Primer

Analyze, research, architect, plan, document

The five-step pipeline is the actual engine behind the template. UNIVERSAL-TEMPLATE-README.md §Workflow Steps Deep Dive (lines 131-219) walks each step with its own time budget, token budget, quality score, and assigned agent. Together they took 22 minutes on the CANVA-CODE validation run and emitted 3,822 lines of implementation-ready documentation.

Step 1 · Analyze Task Space (3 min, ~4,500 tokens, 9.2/10). mercurio-orchestrator runs 14+ sequential-thinking thoughts, decomposes the project into a SPARK vector (Skills, Personality, Approach, Resources, Knowledge), and outputs a constraint map. Step 2 · Research APIs (2 min, ~8,000 tokens, 8.9/10). deep-researcher launches three parallel web searches — Claude SDK, {PRIMARY_API}, {SECONDARY_SERVICES} — and returns a capability-and-limits table. Step 3 · Design Architecture (5 min, ~18,000 tokens, 9.5/10). mars-architect synthesizes the two-agent pattern, MCP tool boundaries, state versioning, and retry logic. This is the step with the highest quality gate in the whole workflow (≥0.90, line 178).

Step 4 · Plan Workflows (4 min, ~12,000 tokens, 9.0/10). mercurio-orchestrator returns and writes three personas — Speed Demon, Perfectionist, Creative Explorer (lines 196-198) — each with primary and iteration flows, measurable success scenarios, and timing breakdowns. Step 5 · Document Everything (8 min, ~13,565 tokens, 9.3/10). docs-generator renders six interconnected markdown files: PROJECT-OVERVIEW.md (248 lines), ARCHITECTURE.md (1,018), API-INTEGRATION.md (1,181), WORKFLOWS.md (564), README.md (386), PROJECT-SUMMARY.md (425). Average across all five steps: 9.18/10. None of the step-scores were aspirational; they are historical measurements from the validation run (lines 423-430).

Five steps, five agents, one sequence. Each has a time budget, a token budget, and a validated quality score. 9.18/10 is the ceiling the template has already hit.
